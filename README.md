# Barcode Detection with YOLOv8

A computer vision application for real-time barcode detection built with **YOLOv8** and deployed using **Streamlit**.

This project trains a custom YOLOv8 model to detect barcodes in images/video streams and provides an interactive web interface for inference.

---

## Features

- Custom barcode detection using YOLOv8
- Streamlit web application for easy interaction
- Training notebook included
- Inference on uploaded images/videos
- Demo recording included
